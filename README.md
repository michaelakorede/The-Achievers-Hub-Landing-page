# Achievers Hub Global Limited Landing Page

Static landing page for Achievers Hub Global Limited.

## Files

- `index.html` - page structure
- `styles.css` - responsive styling
- `script.js` - product finder interaction
- `assets/` - logo, CEO photo, and brand imagery

## GitHub Pages

This site can be hosted from the repository root on GitHub Pages:

1. Create or open a GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings > Pages.
4. Choose Deploy from branch, then select the default branch and `/root`.
5. Save and wait for the Pages URL.
